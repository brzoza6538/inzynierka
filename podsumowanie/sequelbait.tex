\begin{itemize}
    \item Aktualne rozwiązania wykorzystujące algorytmy globalne do sieci neuronowej nie przenoszą informacji o zależności pomiędzy neuronami, wagi są spłaszczane do jednego wymiaru, ignorując powiązania pomiędzy neuronami. Potencjalnie możliwe jest zaprojektowanie algorytmu globalnego dostosowanego specyficznie pod naukę sieci neuronowych, który poradziłby sobie lepiej.
    \item Implementacja kolejnych algorytmów i porównanie ich z obecnymi wynikami. 
    \item Rozszerzenie testów o kolejne benchmarki, takie jak CEC2017.
    \item Badanie hybrydowych metod optymalizacji, łączących algorytmy globalne z lokalnymi lub stochastycznymi.
    \item Opracowanie metod redukcji wymiarów dla zadań optymalizacji sieci neuronowych metodami globalnymi.
    \item Badanie wpływu szumu w danych wejściowych lub funkcji kosztu na skuteczność algorytmów.

\end{itemize}
